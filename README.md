# Проект Mesto фронтенд + бэкенд
Бэкенд проекта Место c регистрацией и авторизацией.

# Используемые технологии
* Node.js
* Express.js 
* MongoDB 
* Mongoose
* Bcrypt
* Celebrate, Joi

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
